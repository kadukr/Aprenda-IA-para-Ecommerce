# IA, Machine Learning, Deep Learning e IA Generativa

Antes de estudar LLM, é importante entender uma confusão comum:

> Machine Learning, Deep Learning e IA Generativa não são “3 IAs separadas”.

Elas são **áreas relacionadas dentro do campo maior da Inteligência Artificial**.

---

## 1. A dúvida principal

Muita gente fala como se existissem três tipos separados de IA:

```text
IA de Aprendizagem de Máquina
IA Profunda
IA Generativa
```

Mas o mais correto é entender assim:

```text
IA
 └── Machine Learning / Aprendizagem de Máquina
      └── Deep Learning / Aprendizagem Profunda
           └── IA Generativa moderna
```

Ou seja:

```text
IA = campo geral

Machine Learning = uma área dentro da IA

Deep Learning = uma área dentro de Machine Learning

IA Generativa = uma área/tipo de modelo que normalmente usa Deep Learning
```

Então elas não são “3 IAs diferentes” no mesmo nível.

Elas são camadas, subáreas e formas de uso.

---

## 2. IA — Inteligência Artificial

IA é o campo maior.

É qualquer sistema que tenta realizar tarefas que parecem exigir inteligência humana.

Exemplos:

```text
entender texto
responder perguntas
reconhecer imagens
tomar decisões
traduzir idiomas
jogar xadrez
recomendar produtos
detectar fraude
gerar texto, imagem, áudio ou vídeo
```

Um sistema pode ser IA mesmo sem usar LLM.

Exemplo:

```text
Um sistema que decide se um pedido parece fraude.
```

Esse sistema pode funcionar de várias formas:

```text
com regras manuais
com Machine Learning
com Deep Learning
com IA Generativa
```

Então:

> IA é o guarda-chuva principal.

---

## 3. Sistemas de IA baseados em regras

Antes de falar de Machine Learning, é importante saber que nem toda IA aprende sozinha.

Existe IA baseada em regras.

Exemplo:

```text
SE valor_do_pedido > 10000
E país_do_cliente for diferente do normal
ENTÃO marcar pedido como suspeito
```

Aqui o sistema não está aprendendo com dados.

Ele apenas segue regras criadas por humanos.

Isso ainda pode ser considerado IA em um sentido mais amplo, mas não é Machine Learning.

Resumo:

```text
IA baseada em regras = humanos escrevem as regras
Machine Learning = o modelo aprende padrões pelos dados
```

---

## 4. Machine Learning — Aprendizagem de Máquina

Machine Learning, ou Aprendizagem de Máquina, é uma área dentro da IA.

Aqui a máquina aprende padrões a partir de dados.

Em vez de você escrever todas as regras manualmente, você dá exemplos para o modelo.

Exemplo:

```text
Pedido 1 → fraude
Pedido 2 → não fraude
Pedido 3 → fraude
Pedido 4 → não fraude
```

Com muitos exemplos, o modelo aprende padrões.

Ele pode aprender que certos comportamentos aumentam o risco de fraude:

```text
valor muito alto
IP diferente do país do cartão
muitas tentativas seguidas
endereço suspeito
e-mail recém-criado
```

Então:

> Machine Learning é IA que aprende com dados.

Exemplos de uso:

```text
prever vendas
detectar fraude
recomendar produtos
classificar clientes
prever cancelamento
reconhecer spam
analisar risco
classificar imagens
```

---

## 5. Deep Learning — Aprendizagem Profunda / IA Profunda

“IA profunda” é uma forma informal de falar de **Deep Learning**.

O termo mais correto em português é:

```text
Aprendizagem Profunda
```

Deep Learning é uma subárea de Machine Learning.

A hierarquia é:

```text
IA
 └── Machine Learning
      └── Deep Learning
```

Deep Learning usa redes neurais com várias camadas.

Por isso é chamado de “profundo”.

---

## 6. O que são redes neurais?

Redes neurais são estruturas matemáticas que recebem dados, processam esses dados em camadas e geram uma saída.

Exemplo simples:

```text
Entrada → Rede Neural → Saída
```

No caso de imagem:

```text
Imagem → Rede Neural → "isso é um gato"
```

No caso de texto:

```text
Texto → Rede Neural → resposta gerada
```

No caso de previsão:

```text
dados de vendas → Rede Neural → previsão de vendas
```

---

## 7. Por que chama Deep Learning?

Chama Deep Learning porque existem várias camadas aprendendo representações diferentes.

Exemplo com imagem de gato:

```text
Entrada: imagem de um gato

Camada 1: detecta linhas
Camada 2: detecta formas
Camada 3: detecta olhos, orelhas e textura
Camada 4: entende que parece um gato
```

Cada camada aprende uma parte mais complexa.

Por isso:

> Deep Learning é Machine Learning usando redes neurais profundas.

---

## 8. IA Generativa — Generative AI

IA Generativa é uma área focada em gerar conteúdo novo.

Ela pode gerar:

```text
texto
imagem
áudio
vídeo
voz
música
código
modelo 3D
```

Exemplo com texto:

```text
Prompt:
Explique o que é Magento 2.

Resposta:
Magento 2 é uma plataforma de e-commerce...
```

Exemplo com imagem:

```text
Prompt:
Crie uma imagem de um robô estudando IA.

Resposta:
Uma imagem nova gerada pelo modelo.
```

Então:

> IA Generativa é IA que cria conteúdo novo.

---

## 9. IA Generativa fica dentro de Machine Learning?

Na prática moderna, sim.

A IA Generativa atual normalmente usa Machine Learning e Deep Learning.

A hierarquia simplificada fica assim:

```text
IA
 └── Machine Learning
      └── Deep Learning
           └── IA Generativa
```

Mas tem um detalhe importante:

IA Generativa é mais uma categoria pelo **tipo de resultado** que ela produz.

Ela é chamada de generativa porque gera algo novo.

Exemplos:

```text
LLM gera texto
modelo de imagem gera imagem
modelo de áudio gera voz ou música
modelo de vídeo gera vídeo
```

---

## 10. Machine Learning vs Deep Learning vs IA Generativa

### Machine Learning

Aprende padrões a partir de dados.

```text
dados → modelo aprende padrões → previsão/classificação/decisão
```

Exemplo:

```text
prever se um cliente vai cancelar
```

---

### Deep Learning

É Machine Learning usando redes neurais profundas.

```text
dados → rede neural profunda → resultado
```

Exemplo:

```text
reconhecer objetos em imagens
entender fala
traduzir texto
criar modelos de linguagem
```

---

### IA Generativa

Usa modelos para criar conteúdo novo.

```text
prompt → modelo generativo → conteúdo novo
```

Exemplo:

```text
gerar texto
gerar imagem
gerar código
gerar áudio
```

---

## 11. Elas estão no mesmo nível?

Não.

Essa é a parte mais importante.

Errado pensar assim:

```text
IA
 ├── Machine Learning
 ├── Deep Learning
 └── IA Generativa
```

Isso dá a impressão de que são três áreas totalmente separadas no mesmo nível.

O mais correto é:

```text
IA
 └── Machine Learning
      └── Deep Learning
           └── IA Generativa moderna
```

Mas também podemos mostrar de uma forma mais completa:

```text
IA
 ├── IA baseada em regras
 ├── Machine Learning
 │    ├── modelos clássicos
 │    │    ├── regressão
 │    │    ├── árvore de decisão
 │    │    └── random forest
 │    │
 │    └── Deep Learning
 │         ├── redes neurais
 │         ├── CNN
 │         ├── RNN
 │         ├── Transformer
 │         └── modelos generativos
 │              ├── LLMs
 │              ├── modelos de imagem
 │              ├── modelos de áudio
 │              └── modelos de vídeo
 │
 └── outras áreas de IA
      ├── busca
      ├── planejamento
      ├── robótica
      └── agentes
```

---

## 12. Onde entram os LLMs?

LLM significa:

```text
Large Language Model
```

Ou em português:

```text
Modelo Grande de Linguagem
```

LLM é um tipo de IA Generativa focado em linguagem.

Ele trabalha com:

```text
texto
perguntas
respostas
resumos
tradução
código
conversas
documentos
```

A hierarquia fica assim:

```text
IA
 └── Machine Learning
      └── Deep Learning
           └── IA Generativa
                └── LLM
```

Mas pelo lado técnico também podemos ver assim:

```text
IA
 └── Machine Learning
      └── Deep Learning
           └── Redes Neurais
                └── Transformer
                     └── LLM
```

As duas formas estão corretas.

Uma mostra **o que o modelo faz**.

A outra mostra **como o modelo é construído**.

---

## 13. Generative AI não é só LLM

LLM é apenas um tipo de IA Generativa.

IA Generativa inclui também:

```text
LLM → gera texto e código
Diffusion Models → geram imagens
modelos de áudio → geram voz, música e som
modelos de vídeo → geram vídeos
modelos 3D → geram objetos tridimensionais
```

Então:

```text
Todo LLM moderno é IA Generativa.
Mas nem toda IA Generativa é LLM.
```

Exemplo:

```text
ChatGPT → LLM / texto
Midjourney → imagem
Suno → música
Runway → vídeo
```

---

## 14. Transformer entra onde?

Transformer é uma arquitetura de rede neural.

Ele não é exatamente uma área separada de IA.

Ele é uma forma de construir modelos.

A hierarquia técnica fica:

```text
IA
 └── Machine Learning
      └── Deep Learning
           └── Redes Neurais
                └── Transformer
                     └── LLM
```

Então:

```text
Transformer = arquitetura
LLM = modelo de linguagem
IA Generativa = categoria de uso
Machine Learning = forma de aprender com dados
Deep Learning = ML com redes neurais profundas
IA = campo geral
```

---

## 15. Exemplo juntando tudo

Imagine o ChatGPT.

Ele é:

```text
IA
porque realiza tarefas inteligentes

Machine Learning
porque foi treinado com dados

Deep Learning
porque usa redes neurais profundas

IA Generativa
porque gera texto novo

LLM
porque é um modelo grande de linguagem

Transformer
porque usa arquitetura Transformer ou variações dela
```

Então uma frase completa seria:

> ChatGPT é uma aplicação baseada em LLM, que é um modelo de IA Generativa, treinado com Machine Learning e Deep Learning, usando redes neurais baseadas em Transformer.

---

## 16. Outro exemplo: modelo de imagem

Imagine um modelo que gera imagens.

Ele é:

```text
IA
porque realiza uma tarefa inteligente

Machine Learning
porque aprendeu com dados

Deep Learning
porque usa redes neurais profundas

IA Generativa
porque cria imagens novas
```

Mas ele não é necessariamente LLM.

Ele pode usar outra arquitetura, como modelos de difusão.

Então:

```text
Modelo de imagem = IA Generativa
Mas não é LLM
```

## 18. Resumo bem simples

```text
IA
é o guarda-chuva.

Machine Learning
é quando a IA aprende com dados.

Deep Learning
é Machine Learning usando redes neurais profundas.

IA Generativa
é quando o modelo cria conteúdo novo.

LLM
é IA Generativa focada em texto e linguagem.

Transformer
é a arquitetura por trás de muitos LLMs.
```

## 19. Ordem correta antes de estudar LLM

Antes de entrar no arquivo sobre LLMs, a ordem ideal é:

```text
1. IA
2. IA baseada em regras
3. Machine Learning
4. Deep Learning
5. Redes Neurais
6. IA Generativa
7. Transformer
8. LLM
```

Depois disso, faz sentido estudar:

```text
tokens
context window
temperature
system prompt
prompt injection
structured output
JSON Schema
function calling / tool calling
embeddings
RAG
fine-tuning
agentes
```
